# [(Binary Min) Heap](./heap.js)

## Time complexity of operations

|              | Time worst case | Time average case |
|--------------|-----------------|-------------------|
| `min`        | `O(1)`          | `O(1)`            |   
| `insert`     | `O(log(n))`     | `O(1)`            |
| `extractMin` | `O(log(n))`     | `O(log(n))`       |
| heapify      | `O(n)`          | `O(n)`            |

## Memory

`O(n)`
